# ADR-0003-kill-ai-strategies

Date: 2025-09-28

## Status
KILLED

## Decision
Автоматическое создание стратегий через ИИ исключено из Scarlet Sails.

## Context
В проекте KIT_RnD_Layer4 была идея использования ИИ для автоматической генерации торговых стратегий на основе исторических данных и рыночных условий. Подход включал генетические алгоритмы, reinforcement learning и автоматическую оптимизацию параметров.

## Problem
Потеря контроля над торговой логикой создает неприемлемые риски. ИИ-генерированные стратегии могут работать на исторических данных, но непредсказуемо вести себя в реальных рыночных условиях. Отсутствие объяснимости решений делает невозможным риск-менеджмент.

## Consequences
ИИ используется только как помощник для анализа данных и генерации сигналов. Все торговые решения остаются под контролем человека. Стратегии создаются и тестируются вручную с использованием ИИ для вспомогательного анализа.

## Alternatives Considered
Рассматривались варианты ограниченного использования ИИ для оптимизации параметров существующих стратегий, но даже это создает риски неконтролируемого поведения в нестандартных рыночных условиях.

## Lessons Learned
ИИ должен оставаться инструментом, а не заменой человеческого контроля в критически важных решениях. Торговые стратегии требуют понимания рыночной логики, которую невозможно полностью автоматизировать.
