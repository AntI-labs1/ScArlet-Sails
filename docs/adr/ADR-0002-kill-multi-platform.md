# ADR-0002-kill-multi-platform

Date: 2025-09-28

## Status
KILLED

## Decision
Параллельная торговля на множественных платформах исключена из первой версии Scarlet Sails.

## Context
В проекте KIT_RnD_Layer4 была попытка одновременной интеграции с множественными торговыми платформами:
- Binance
- Bybit
- OKX
- Kraken
- Локальные DEX

## Problem
- Сложность интеграции превышает выгоду
- Различные API и форматы данных
- Сложность синхронизации состояния
- Увеличение поверхности для ошибок
- Преждевременная оптимизация

## Consequences
- Фокус на одной платформе (Binance) для MVP
- Архитектура позволяет добавление платформ в будущем
- Унифицированный интерфейс для торговых операций
- Масштабирование только после доказательства концепции

## Alternatives Considered
1. Универсальный адаптер - отклонено, слишком сложно
2. Поэтапное добавление платформ - отклонено для MVP
3. Фокус на одной платформе - принято

## Lessons Learned
- Сначала одна платформа, потом масштабирование
- MVP должен решать одну задачу хорошо
- Архитектура должна позволять расширение без переписывания
- Доказать концепцию перед масштабированием
