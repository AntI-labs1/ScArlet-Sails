# ADR-0001-kill-complex-cnn

Date: 2025-09-28

## Status
KILLED

## Decision
Сложная многоуровневая CNN архитектура исключена из ядра Scarlet Sails.

## Context
В проекте KIT_RnD_Layer4 была предпринята попытка создания сложной многоуровневой CNN архитектуры для анализа рыночных паттернов. Архитектура включала:
- Множественные сверточные слои
- Сложные механизмы attention
- Ensemble методы
- Автоматическую генерацию признаков

## Problem
- Overengineering без доказанной эффективности
- Высокая сложность разработки и поддержки
- Отсутствие baseline для сравнения
- Невозможность объяснить принятые решения
- Высокие требования к вычислительным ресурсам

## Consequences
- Функции демонтированы из основного кода
- Остается sandbox track для исследований
- Приоритет отдается простым моделям с доказанной эффективностью
- Обязательное сравнение с baseline перед внедрением ML

## Alternatives Considered
1. Упрощение архитектуры - отклонено, так как базовая проблема в подходе
2. Поэтапное внедрение - отклонено, нет доказательств эффективности
3. Полное исключение - принято

## Lessons Learned
- Начинать с простых моделей (линейная регрессия, простые классификаторы)
- Всегда иметь baseline для сравнения
- ML должен улучшать существующую логику, а не заменять её
- Объяснимость важнее сложности
